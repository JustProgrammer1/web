O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar e estruturar o conteúdo de páginas web. Não é uma linguagem de programação — ela não executa lógica nem processa dados. Em vez disso, utiliza tags (marcações entre colchetes angulares) para indicar ao navegador como cada parte do conteúdo deve ser interpretada: títulos, parágrafos, imagens, links, listas, tabelas, e muito mais.

Qual o papel do HTML na Web?

Na arquitetura da Web, o HTML é a base de tudo. Quando você acessa um site, o navegador (cliente) faz uma requisição HTTP ao servidor, que devolve um arquivo HTML. O navegador então lê esse arquivo e renderiza o conteúdo na tela. O HTML define o quê aparece na página; o CSS define como aparece; e o JavaScript define o que acontece quando o usuário interage. Juntos, esses três formam a tríade fundamental do desenvolvimento web.

Principais tags semânticas do HTML5

  • header — Cabeçalho da página ou seção
  • main — Conteúdo principal da página
  • footer — Rodapé da página ou seção
