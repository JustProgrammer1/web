Questão 2 — Estrutura e Sintaxe do CSS

O que é uma regra CSS?

Uma regra CSS é formada por três partes:

  • Seletor: indica qual(is) elemento(s) HTML será(ão) estilizados. Exemplos: p, .classe, #id.
  • Propriedade: o aspecto visual a ser alterado. Exemplos: color, font-size, background-color.
  • Valor: define como a propriedade se comporta. Exemplos: red, 16px, #ffffff.
