Protocolo HTTP

HTTP é o protocolo que permite a comunicação entre o navegador e o servidor. Quando você acessa um site, seu navegador faz um pedido (request) e o servidor responde (response).

Exemplo com formulário







Ao enviar, o navegador faz uma requisição HTTP.

Exemplo de Request

GET /?nome=Ana&mensagem=Oi HTTP/1.1

Exemplo de Response

HTTP/1.1 200 OK

Explicação dos campos

Request:

  • GET: tipo de requisição
  • URL: endereço solicitado
  • Parâmetros: dados enviados (nome, mensagem)

Response:

  • Status 200: requisição deu certo
  • Conteúdo: resposta do servidor (página HTML)
