Diferença de Internet e Web

A Internet é a infraestrutura física global(cabos, servidores, roteadores) que conecta computadores mundialmente. A Web (World Wide Web) é o sistema de páginas, documentos e hipermídia que trafega por essa rede, acessado via navegadores. Resumindo: a Internet é a rede física, enquanto a Web é o conteúdo/aplicação que usamos nela. Sem a Internet, a Web não funciona.
